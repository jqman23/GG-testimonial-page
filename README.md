# Global Gathering Testimonial Page

A responsive static page presenting participant reflections from positive, general Global Gathering feedback.

## Files

- `index.html` - page markup
- `styles.css` - responsive widget styling
- `script.js` - featured reflection controls and auto-rotation
- `data/testimonials.json` - reviewed source data loaded by the page

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server:

```bash
npm start
```
